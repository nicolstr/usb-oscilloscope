1. Install python3 interpreter (working version example: Python 3.13.2).
   
   Make sure to check option Add Python 3.5 to PATH.

2. Install required python packages with pip:
      
         bash: pip install -r .\requirements.txt
         
      (
         may need to be executed more than once to ensure clean installation
         
         check successful installation of a package:
         bash:    pip show <package_name>
      )
      
3. Start application:
         bash:    py .\oszi_oberflaeche.py
